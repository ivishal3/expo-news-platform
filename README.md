## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npm start
   ```

3. EAS Login

   ```bash
   eas login
   ```

4. Build the app

   ```bash
   eas build --platform android --profile preview
   ```
